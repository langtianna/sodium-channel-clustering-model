# Sodium Channel Clustering Model

Two-compartment Hodgkin–Huxley model exploring tradeoffs between spike coordination and propagation speed.

## Associated manuscript

Lang, T. (2026)  
*A Minimal Conductance-Based Model Reveals a Coordination–Propagation Tradeoff Under Conserved Sodium Redistribution*

## Description

This code implements a minimal two-compartment neuron model in which total sodium conductance is conserved and redistributed between compartments.

The model is used to analyze:
- spike initiation
- spike propagation
- coordination between compartments
- the coordination–propagation tradeoff
